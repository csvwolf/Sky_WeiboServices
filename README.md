## 程序说明

### script说明

一个方便设置Crontab的脚本，使用Python2.7编写，需要`pip install plan`，目前还不太好用，待扩展/重构。
执行可以查看到使用帮助。

### 主程序说明

index.js是该程序的主入口，如果你需要设置定时，请启动`script.py`。
在index.js中，使用`node index.js command [more...]`来运行。


### 目录结构介绍

- actions：基础动作，即封装调用API
- config：配置文件（包括路由）
- controllers：功能模块
